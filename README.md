## A Little Note

你好，我是 **筱柒 Littleseven**。

我会对日常里那些「要是能少点几下就好了」的瞬间格外在意。

所以这里大多是一些从真实需求里长出来的小项目：给牌局记分、整理工时、处理表格，或者研究一份感兴趣的数据。它们未必复杂，但最好真的顺手，真的能替人省下一点时间。

```text
notes from my desk
├── automate the repetitive
├── polish the everyday
├── stay curious about data
└── make first, learn along the way
```
